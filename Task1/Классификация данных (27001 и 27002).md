1. **Публичные данные (Public)**
   - Открытая информация о недвижимости, доступная на витрине (адрес, общие характеристики ЖК).
   - Описание акций, рекламные материалы, новости на сайте.

2. **Внутренние данные (Internal)**
   - Регламенты, инструкции для сотрудников.
   - Статистика по продажам в обобщённом виде (без персональных данных).
   - Внутренние отчёты по работе IT-систем (лог-файлы без критичных данных).
   - Информация о статусах разработки (беклог, план релизов).

3. **Конфиденциальные данные (Confidential)**
   - Персональные данные клиентов и собственников (ФИО, телефон, e-mail, паспортные данные, данные о собственности).
   - Коммерческие условия сделок (цены, скидки, условия рассрочки/ипотеки).
   - Финансовые данные компании (отчётность, бухгалтерские данные), не попадающие в публичные сводки.
   - Информация о интеграционных контрактах с партнёрами, договорах с госорганами.

4. **Секретные данные (Secret)**
   - Критически важные данные и сервисы, дающие конкурентное преимущество: алгоритмы ценообразования, стратегические планы, M&A-проекты.
   - Ключи шифрования, пароли к внутренним системам.
   - Учётные данные и схемы доступа к финансовым транзакциям.
