| Роль   | Права роли                                                                 | Группы пользователей |
| ------ | -------------------------------------------------------------------------- | -------------------- |
| admin  | Полный доступ (CRUD) к объектам внутри namespace, включая просмотр секретов | Группа DevOps и старшие админы |
| editor | Возможность настраивать ресурсы внутри namespace (Pods, Deployments, ConfigMaps и т. д.) | Группа разработчиков/операторов |
| viewer | Просмотр (read-only) ресурсов внутри namespace (Pods, Services, Deployments, ConfigMaps) | Группа аналитиков/наблюдателей |
